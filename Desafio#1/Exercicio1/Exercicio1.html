Considere a transformao da varivel que definida por ... onde dfi o nmero de documentos em que o i-simo termo aparece e conhecido como a frequncia do documento do termo. Essa transformao conhecida como transformao inversa da frequncia do documento

a) Qual o efeito dessa transformao se um termo ocorrer em um documento? E em todos os documentos?

Se o termo ocorre em apenas um documento sua frequencia ser multiplicada pelo log do numero de documentos, desta forma este termo ser destacado por ser mais raro. Caso contrario se o termo ocorre em todos os documentos ento dfi = m, desta forma sua frequncia ser multiplicada pelo log de 1 o que anula sua frequncia.

b) Qual pode ser o propsito dessa transformao?

Esta transformao destaca termos que so mais 'raros' que devem ser considerados mais relevantes quando uma busca for realizada.

import math

def med(x):
    soma = 0
    for i in range(0, len(x)):
        soma += x[i]
    med = soma/len(x)
    return(med)

def cov(x, y):
    medX = med(x)
    medY = med(y)
    soma = 0
    for i in range(0, len(x)):
        soma += ((x[i] - medX) * (y[i] - medY))
    cov = soma/(len(x)-1)
    return(cov)

def sdev(x):
    medX = med(x)
    soma = 0
    for i in range(0, len(x)):
        soma += ((x[i] - medX) * (x[i] - medX))
    sdev = math.sqrt(soma/(len(x)-1))
    return(sdev)
    
def cos(x, y):
    prod = 0
    magX = 0
    magY = 0
    for i in range (0, len(x)):
        prod += x[i]*y[i]
        magX += x[i]*x[i]
        magY += y[i]*y[i]
    magX = math.sqrt(magX)
    magY = math.sqrt(magY)
    cos = prod/(magX*magY)
    return(cos)

def corr(x, y):
    covXY = cov(x, y)
    sdevX = sdev(x)
    sdevY = sdev(y)
    if(sdevX == 0 or sdevY == 0):
        return("Undefined")
    corr = covXY/(sdevX*sdevY)
    return(corr)

def euc(x, y):
    soma = 0
    for i in range (0, len(x)):
        soma += (x[i] - y[i])*((x[i] - y[i]))
    euc = math.sqrt(soma)
    return(euc)

def jac(x, y):
    f01 = 0
    f10 = 0
    f11 = 0
    for i in range (0, len(x)):
        if(x[i] == 0 and y[i] == 1):
            f01 += 1
        elif(x[i] == 1 and y[i] == 0):
            f10 += 1
        elif(x[i] == 1 and y[i] == 1):
            f11 += 1
    jac = f11/(f01+f10+f11)
    return(jac)

4 - Discutir as diferenas entre a reduo da dimensionalidade com base em agregao e da reduo da dimenso com base em tcnicas como PCA.

Redues com base em agragao combinam 2 ou mais atribuitos em um unico atributo. Agregaes no tem como abjetivo apenas a reduo de dimenes mas tambm mudanas de escala bem como formataes mais 'estaveis' dos dados. Redues com base em tcnicas como PCA tem como objetivo apenas a reduo de dimenes. O PCA, por exemplo, apenas busca achar uma projeo que capture a maior variao nos dados, no sem importando com escala ou formatao dos dados.
