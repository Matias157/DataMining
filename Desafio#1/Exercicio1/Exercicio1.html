1 - Considere uma matriz esparça (document-term matrix), em que tfij é a frequência da i-ésima palavra (termo) no j-ésimo documento e m é o número de documentos. Considere a transformação da variável que é definida por ... onde dfi é o número de documentos em que o i-ésimo termo aparece e é conhecido como a frequência do documento do termo. Essa transformação é conhecida como transformação inversa da frequência do documento

a) Qual é o efeito dessa transformação se um termo ocorrer em um documento? E em todos os documentos?

Se o termo ocorre em apenas um documento sua frequencia será multiplicada pelo log do numero de documentos, desta forma este termo será destacado por ser mais raro. Caso contrario se o termo ocorre em todos os documentos então dfi = m, desta forma sua frequência será multiplicada pelo log de 1 o que anula sua frequência.

b) Qual pode ser o propósito dessa transformação?

Esta transformação destaca termos que são mais 'raros' que devem ser considerados mais relevantes quando uma busca for realizada.

import math

def med(x):
    soma = 0
    for i in range(0, len(x)):
        soma += x[i]
    med = soma/len(x)
    return(med)

def cov(x, y):
    medX = med(x)
    medY = med(y)
    soma = 0
    for i in range(0, len(x)):
        soma += ((x[i] - medX) * (y[i] - medY))
    cov = soma/(len(x)-1)
    return(cov)

def sdev(x):
    medX = med(x)
    soma = 0
    for i in range(0, len(x)):
        soma += ((x[i] - medX) * (x[i] - medX))
    sdev = math.sqrt(soma/(len(x)-1))
    return(sdev)
    
def cos(x, y):
    prod = 0
    magX = 0
    magY = 0
    for i in range (0, len(x)):
        prod += x[i]*y[i]
        magX += x[i]*x[i]
        magY += y[i]*y[i]
    magX = math.sqrt(magX)
    magY = math.sqrt(magY)
    cos = prod/(magX*magY)
    return(cos)

def corr(x, y):
    covXY = cov(x, y)
    sdevX = sdev(x)
    sdevY = sdev(y)
    if(sdevX == 0 or sdevY == 0):
        return("Undefined")
    corr = covXY/(sdevX*sdevY)
    return(corr)

def euc(x, y):
    soma = 0
    for i in range (0, len(x)):
        soma += (x[i] - y[i])*((x[i] - y[i]))
    euc = math.sqrt(soma)
    return(euc)

def jac(x, y):
    f01 = 0
    f10 = 0
    f11 = 0
    for i in range (0, len(x)):
        if(x[i] == 0 and y[i] == 1):
            f01 += 1
        elif(x[i] == 1 and y[i] == 0):
            f10 += 1
        elif(x[i] == 1 and y[i] == 1):
            f11 += 1
    jac = f11/(f01+f10+f11)
    return(jac)

4 - Discutir as diferenças entre a redução da dimensionalidade com base em agregação e da redução da dimensão com base em técnicas como PCA.

Reduções com base em agragação combinam 2 ou mais atribuitos em um unico atributo. Agregações não tem como abjetivo apenas a redução de dimenções mas também mudanças de escala bem como formatações mais 'estaveis' dos dados. Reduções com base em técnicas como PCA tem como objetivo apenas a redução de dimenções. O PCA, por exemplo, apenas busca achar uma projeção que capture a maior variação nos dados, não sem importando com escala ou formatação dos dados.
