Problema Escolhido¶

O problema escolhido para este trabalho foi a identificao de similaridade de sons de lnguas. A ideia analizar e enterder quais os fatores, ou caracteristicas, em comum duas linguas devem apresentar a fim de que soem similares.

Fontes de dados¶

Os principais dados utilizados vieram de trs fontes distintas. A primeira foram os dados abertos a respeito das jogadas do "The Great Language Game", onde neste jogo o jogador ouvia diversas frases faladas em linguas diferentes, com o objetivo de adivinhar qual lingua estava sendo falada, e lhe eram dadas quatro opes de linguas para escolher. A segunda foi a base de dados aberta World Atlas of Language Structures (WALS), da qual foram retiradas informaes a respeito da geografia das linguas, bem como suas fimlias linguisticas. Por fim foi utilizada a base de dados aberta Phonetics Information Base and Lexicon (PHOIBLE), da qual foi possivel encontrar os fonemas das linguas que estavam sendo analisadas.

Agora com uma prpria leitura dos dados foi realizada uma segunda formao, esta que vai servir como base para todo o desenvolvimento do projeto. As linguas foram agruapadas em pares e foi calculada a probablidade da lingua target ser confundida com a lingua guess de duas formas diferentes. A primeira foi contar quantas vezes a lingua target era confundida com a guess e divir este valor pelo total de vezes que a lingua target apareceu no jogo. O segundo calculo foi similar ao primeiro, porm neste caso foi dividido pelo total de vezes que a ligua target foi confunda com qualquer lingua (foram excluidas as jogadas de acerto). Aps anlise foi decidido por ir em frente apenas com o segundo calculo, visto que ele faz mais sentido com os objetidos do projeto.

Agora com os dados das probabilidades necessrio binariza-los, onde o par recebe 1 se a linguagem target tem uma maior change de ser confundida com a lingua guess e recebe 0 caso contrrio. Como as probabilidades apresentam valores baixos (a maior calculada foi aproximadamente 8%) foi feita a utilizao de uma Funo de Distribuio Acumulada (FDA ou CDF) para descobrir o valor de probabilidade que apresenta a probabilidade acumulada de 50%, e partir deste valor ser feita a binarizao dos dados.

Resultados¶

Finalizando a formatao dos dados utilizado uma estratgia de rvore de deciso com campos obtidos anteriomente para tentar classisfic-los.

Os campos que alimentaro o algoritmo so as macroreas de ambas as lnguas target e guess, o nmero de fonemas em comum que as lnguas apresentam, suas famlias lingusticas e suas coordenadas geogrficas.
